Приложение и возможности 3CX

4.03.2022

3CX позволяет коммуницировать из любого места в мире благодаря мобильным приложениям для Android / iOS, софтфонам для Windows и браузера. Теперь вы всегда включены в работу! А клиенты могут связаться с вами в любой момент через коммуникационный виджет для сайта 3CX Live Chat или интеграцию с Facebook Messenger.

Приложения 3CX для устройств Android и iOS
  • Совершайте и принимайте вызовы в любой точке мира
  • Создавайте видеоконференции
  • Внутренний чат и сообщения SMS
  • Отвечайте на сообщения с бизнес-страницы Facebook
  • Голосовые конференции
  • Бесплатные VoIP-вызовы со смартфона
  • Шифрование и безопасность
Удобный веб-клиент / приложение Windows Desktop App
  • Удаленная работа отовсюду — многоцелевое веб-приложение с мощными возможностями UC
  • Звоните прямо из браузера, управляйте вызовами с IP-телефона или смартфона
  • Отвечайте на обращения пользователей через чат для сайта Live Chat, Facebook и SMS — в
    едином удобном интерфейсе
  • Интегрированные видеоконференции в один клик
  • Нативное приложение Windows Desktop App для управление вызовами
  • Делайте вызовы прямо из CRM-системы
  • Фирменная интеграция с MS Teams и MS 365
  • Индикация статусов / присутствия сотрудников
3CX Live Chat — коммуникации нового поколения для вашего сайта
  • Общайтесь с посетителями вашего сайта
  • Создавайте очереди обработки входящих чатов
  • Отвечайте через веб-клиент или приложение 3CX
  • Переводите чаты другим сотрудникам
  • Переводите чат в голосовой или видеовызов одним кликом
  • Различные отчеты о чатах
  • Функции мониторинга чатов для руководителей!
Интеграция с Facebook Messenger
  • Интеграция с бизнес-страницей FB — все сообщения приходят в веб-клиент 3CX
  • Операторы подключены к очередям сообщений без доступа в Facebook
  • Направляйте сообщения в очередь для группы операторов
  • Отчеты о чатах по очередям и отдельным операторам
Отправляйте и принимайте SMS / MMS для бизнеса
  • Отправляйте SMS / MMS вашим клиентам
  • Направляйте SMS на группу операторов, чтобы ни одно обращение не было потеряно
  • Операторам не нужно раздавать свой личный мобильный номер
